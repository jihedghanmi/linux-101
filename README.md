# Linux 101 — Learning Journey

A hands-on Linux learning repository documenting my daily practice with 
real sysadmin tasks, bash scripting, and system administration concepts.

## What's inside

| Script | Description |
|--------|-------------|
| `append_log.sh` | Logs "server started" with timestamp to a log file |
| `monitor.sh` | Displays top 5 processes by memory consumption |
| `network_info.sh` | Shows local IP, public IP, and internet connectivity status |

## Topics covered so far

- User and group management
- File permissions and ownership (chmod, chown)
- Process management (ps, kill, background jobs)
- Network diagnostics (ip, ss, ping, curl)
- Bash scripting (variables, if/else, functions, append)
- Git and GitHub workflow

## Stack

- OS: Fedora Linux
- Shell: Bash
- Version control: Git
